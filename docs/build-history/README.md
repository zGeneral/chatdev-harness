# Build history (archived)

These are **point-in-time records of the original autonomous build** (the Ralph-loop that first
created the harness). They describe an earlier shape that has since been consolidated onto a single
**declarative graph engine**, so they reference commands/workflows that no longer exist
(`/build-company`, `/build-game`, `chatdev-company.js`, `chatdev-gamedev.js`).

Kept for provenance only — for the **current** model see `README.md` and `CLAUDE.md`.

- `PROGRESS.md` — the original iteration log (Phases A–D).
- `BUILD_PLAN.md` — the original design/plan.
- `HARNESS_DONE.md` — the original completion marker.
